# PKU Thesis Download - 北大论文下载插件

Chrome 浏览器扩展，用于下载[北京大学学位论文数据库](https://thesis.lib.pku.edu.cn/)中可查看的论文 PDF。

## 功能

- 一键下载论文所有页面，自动合并为 PDF 文件
- 支持多种访问方式：直连、Web VPN、PAC VPN、北医 WebVPN
- 自动提取作者名作为文件名，支持手动修改
- 下载完成后自动在文件管理器中显示文件
- 优化论文阅读体验：预加载、加载失败自动重试
- 附带 OCR 转换工具，可将图片 PDF 转为可搜索文字的 PDF

## 安装教程（3 步完成）

### 第 1 步：下载插件

点击下面的链接，下载插件压缩包：

> **👉 [点击这里下载 ZIP](https://github.com/Aspirin404/PKU-Thesis-Download/archive/refs/heads/main.zip)**

下载完成后，**解压缩** ZIP 文件，会得到一个名为 `PKU-Thesis-Download-main` 的文件夹。

> 记住这个文件夹的位置（建议放在桌面或下载目录），后面要用到。

### 第 2 步：打开 Chrome 扩展管理页面

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/`，按回车
3. 找到页面**右上角**的「**开发者模式**」开关，**打开它**

### 第 3 步：加载插件

1. 点击页面左上角的「**加载已解压的扩展程序**」按钮
2. 在弹出的文件选择窗口中，找到并选择刚才解压出来的 **`PKU-Thesis-Download-main`** 文件夹
3. 点击「**选择**」

看到扩展列表中出现「**PKU Thesis Download - 北大论文下载**」就说明安装成功了！

> **提示**：建议点击 Chrome 工具栏右侧的拼图图标 🧩 → 找到本扩展 → 点击📌固定，这样图标会常驻在工具栏。

## 使用方法

1. 打开北大学位论文数据库，找到要下载的论文，点击「**查看全文**」进入阅读页面
2. 页面**右下角**会出现一个**红色圆形下载按钮**，点击它
3. 在弹出的下载面板中，可以修改文件名
4. 点击「**开始下载**」，等待进度条完成
5. 下载完成后，文件管理器会自动打开并高亮显示下载的 PDF 文件

> 如果右下角没有出现红色按钮，点击浏览器工具栏的扩展图标 → 点击「**手动注入脚本**」。

## OCR 文字识别（可选，macOS）

下载的 PDF 是图片格式（无法选中文字）。如果需要可搜索、可复制文字的 PDF，可以用附带的 OCR 工具转换：

**安装 OCR 工具（只需一次）：**

打开终端（Terminal），运行：
```bash
cd ~/Desktop/PKU-Thesis-Download-main   # 进入插件目录（根据你的实际位置修改）
bash setup_ocr.sh
```

**转换论文：**
```bash
python3 ocr_convert.py ~/Downloads/你的论文.pdf
```

转换完成后，同目录下会生成 `你的论文_OCR.pdf`，外观不变，但文字可选中、可搜索。

## 常见问题

**Q: 安装后看不到下载按钮？**

A: 点击 Chrome 工具栏的扩展图标（拼图 🧩）→ 找到「PKU Thesis Download」→ 点击「手动注入脚本」。

**Q: 下载中途提示 403 错误？**

A: 这是服务器限流，插件会自动重试。如果重试多次仍失败，请等几分钟再试。

**Q: 文件名显示为一串乱码？**

A: 论文阅读页面不包含标题信息，插件会自动提取作者名。你可以在下载面板中手动修改文件名后再点击下载。

**Q: 下载的 PDF 文字无法选中/复制？**

A: 北大论文数据库只提供图片格式（DRM 保护）。使用上方的 OCR 工具可以添加文字层。
